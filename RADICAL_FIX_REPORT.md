- ✅ إضافة `cache-control: immutable` لتخزين الأيقونات دائماً

**الملفات المعدلة:**

- ✅ `public/icon-192x192.svg` - أيقونة جديدة

- ✅ `next.config.js` - إضافة headers للأيقونات

### 2. ❌ خطأ 413 (Content Too Large)

**المشكلة الأصلية:**
```
[Compress] Status: 413
Error: Content Too Large
POST https://an-academy.vercel.app/api/compress-pdf 413 (Content Too Large)
```

**السبب:**
- حد الملفات المسموح (50MB) كان قليل جداً
- لم يكن هناك معالجة صحيحة للأخطاء

**الحل الجذري:**
- ✅ زيادة `bodySizeLimit` من 100mb إلى **300MB** في next.config.js
- ✅ زيادة `maxDuration` من 60 ثانية إلى **120 ثانية** في API
- ✅ زيادة حد الملفات من 50MB إلى **100MB** في الـ API
- ✅ إضافة معالجة محددة لخطأ 413 مع رسالة واضحة
- ✅ إضافة تتبع حجم الملفات بصيغة MB بدل bytes

**الملفات المعدلة:**
- ✅ `next.config.js`
  ```js
  bodySizeLimit: '300mb'  // كان: 100mb
  allowedOrigins: [..., 'an-academy.vercel.app', '*.vercel.app']
  ```
- ✅ `src/app/api/compress-pdf/route.ts`
  ```ts
  maxDuration = 120  // كان: 60
  MAX_SIZE = 100 * 1024 * 1024  // كان: 50 * 1024 * 1024
  ```
